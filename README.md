# AI Developer 101 Guide

## A Comprehensive Training Guide for Non-Computer Science Backgrounds

Welcome to the AI Developer 101 Guide! This course is designed specifically for individuals coming from streams other than computer science who want to learn how to build AI-powered applications from scratch.

---

## Who Is This Guide For?

- Business professionals wanting to understand AI development
- Students from non-technical backgrounds
- Domain experts looking to integrate AI into their work
- Anyone curious about how AI applications are built

---

## What You Will Learn

By the end of this guide, you will be able to:

1. Write Python programs from scratch
2. Design and query databases
3. Build REST APIs that connect frontend and backend
4. Understand how Large Language Models (LLMs) work
5. Integrate AI models into your applications
6. Build knowledge bases with vector databases
7. Implement RAG (Retrieval Augmented Generation)
8. Create AI agents using LangChain and LangGraph
9. Build a complete AI-powered travel assistant

## Prerequisites

- A computer with internet access
- Willingness to learn
- No prior programming experience required!

---

## How to Use This Guide

1. **Follow the chapters in order** - Each chapter builds upon the previous one
2. **Type out the code yourself** - Don't just copy-paste; typing helps you learn
3. **Complete the practice exercises** - They reinforce your understanding
4. **Experiment** - Modify the examples and see what happens
5. **Ask questions** - Use AI assistants like ChatGPT or Claude to clarify doubts

---

## Setting Up Your Learning Environment

Before starting, you'll need:

1. **Python 3.10+** - We'll guide you through installation in Chapter 1
2. **VS Code or PyCharm** - Your code editor (free versions available)
3. **PostgreSQL** - Database system (covered in Chapter 2)
4. **API Keys** - For OpenAI, etc. (covered in respective chapters)
